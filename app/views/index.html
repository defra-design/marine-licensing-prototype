Multiple sites v2 Ready for review

This version provides enhanced support for multiple sites, allowing users to provide coordinates through file upload and manual entry within the same exemption journey. You can add and delete sites multiple times using different methods.

Start from:

  • The 'You are exempt' page of the 'Check if you need a licence' section
  • Task list with project name completed
  • The beginning of the IAT replica journey

This is a prototype. You are not required to upload a file – just select 'Continue' on the upload page. The mapped coordinates shown are generic.

When testing file uploads, the sequence is: first upload contains 4 sites, second upload contains 1 site, and all subsequent uploads contain 2 sites.

A possible addition

On the 'Your sites' page add the option to select multiple sites and delete them in one go.

Interactive Assistance Tool (IAT) Finished

Start the IAT journey

Start the mock IAT journey

This is a replica of the live Interactive Assistance Tool (IAT) which helps users determine whether they need a marine licence. The tool guides users through a series of questions to assess their proposed marine activities.

What each asset is doing

Technical architecture

Asset Role in the prototype
app/data/iat.json Single source of truth for content & routing.
The route file (iat.js) reads this JSON to build every question/outcome page at runtime.
iat.js (routes) Converts the JSON into Express routes, decides whether to render a radio-page, card-selection, or outcome.
.njk templates Present the pages: radio-page, card-selection, outcome, and (now) start.html.
IAT-design-spec.md Human-readable spec – lists every screen's copy, component, and logic so you (and reviewers) can understand / QA the journey. Not referenced by any code.
iat-tech-spec.csv / flow diagram Additional documentation aids; likewise not used programmatically.

Iteration 4

Start the iteration 4 journey

v4 now features a dashboard 'Your projects', and asking 'Project name' before landing on the task list page. Read about all the changes.

The scenarios to follow for this iteration

We are planning to test with novice and professional users, so both scenarios are relevant for this iteration.

Scenario 1

You're looking to carry out some research involving analysis of sediments from just off the coast of Worthing, by the pier.

You need to carry out a survey taking 1 sample within 100m of the pier.

You'll travel to the sample sites by boat and lower a grab overboard.

The grab will be used to extract 0.3m3 of material from the surface of the sea floor.

The material removed will be returned to the vessel and taken anyway for testing and analysis.

The material will be disposed to land once testing and analysis is complete.

A colleague advised that you might need to get permission from the Marine Management Organisation before you carry out this activity.

Scenario 2

You live in Hammersmith London W6 9TS with a private garden fronting onto the River Thames.

You'd like to install a new private pontoon of 20m2, accessible via gangway, enabling independent access to the River Thames. You'll employ a builder to do this work for you.

Five of your neighbours properties have a similar pontoon.

The Pontoon and gangway will be constructed and installed on site.

You have also applied for permission from the Marine Harbour Authority.

Works are planned to be conducted in one phase, over an estimated period of 2 days in the period April-May 2025.
