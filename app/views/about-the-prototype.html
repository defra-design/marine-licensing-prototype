This prototype is made using the GOV.UK Prototype Kit and components and patterns from the the GOV.UK Design System.

User journeys in this prototype

  • Check if you need a marine licence
  • The exemption notification journey

The 'Check if you need a marine licence' journey guides users through a series of questions and helps them determine whether their marine activity requires a licence or is exempt.

Exempt activities do not require a marine licence but MMO need to be notified of the activity.

Drawing on a map

For drawing a location/site on a map, the prototype uses the map software that the 2024 alpha were using and developing. This has some limitations in terms of usability and accessibility and we have been investigating other fully accessible and more usable options that could be used instead, including the Flood risk map that is being developed within Defra.

Prototype limitations

If you see a page titled 'Journey end' it means either a feature you've selected isn't available or you need to go back and answer a question differently.

Clearing session data

For the prototype to function and display correctly, the data entered during a session is saved. If you revisit a journey without clearing this data, the prototype will not function correctly. To ensure it does, you must clear the data each time. You can do this using the 'Clear data' link in the footer and on the index page.

Iteration 1

The 'Check if you need a licence' journey that allows for the scenario relating to removal of sediment from the sea bed for scientific research. When the questions have been successfully answered the 2nd part is the exemption notification journey where details of the activity are entered.

It's a linear journey with no account creation or sign in required.

The 'Check if you need a licence' journey is an existing live service, so our Content Designer reviewed the questions to ensure they align with the Government Digital Service style guide, restructuring them where necessary. Some questions were also removed because they repeated information users had already provided. Read the content change log.

The Marine Protected Areas (MPA) question links to an arcgis mapping tool which has various layers on it to help identify different areas. This is linked to from the live service, but in the prototype the map loads with only the MPA layer visible, rather than all of them, which is currently the case in live.

The available location details options are draw on a map and specify the centre of a square and its width.

Scenario 1 details

You're looking to carry out some research involving analysis of sediments from just off the coast of Worthing, by the pier.

You need to carry out a survey taking 1 sample within 100m of the pier.

You'll travel to the sample sites by boat and lower a grab overboard.

The grab will be used to extract 0.3m3 of material from the surface of the sea floor.

The material removed will be returned to the vessel and taken anyway for testing and analysis.

The material will be disposed to land once testing and analysis is complete.

A colleague advised that you might need to get permission from the Marine Management Organisation before you carry out this activity.

Iteration 2

Based on the UR feedback some of the 'Check if you need a marine licence' questions were updated with different or new content, but we also added in new pages to allow us to test a different scenario more appropriate for novice users - pontoon construction in a residential setting.

Features added:

  • File with coordinates upload options, with KML file upload the one that works
  • Specifying centre of a circle and it's width as the location
  • Manually entering multiple coordinates of the area
  • Specifying which coordinate system is used for coordinate entry (WGS84 or OSGB36)
  • 'Check answers' and 'Review your location' now uses Cards style summary lists, with location changes happening only on the 'Review your location' page with a single link to that page from 'Check answers'

We didn't have time to iterate all the recommendations from all the UR feedback (specifically dates guidance and duplication of project and activity details), as we were waiting for a response from the legal team.

Scenario 2 details

You live in Hammersmith London W6 9TS with a private garden fronting onto the River Thames.

You'd like to install a new private pontoon of 20m2, accessible via gangway, enabling independent access to the River Thames. You'll employ a builder to do this work for you.

Five of your neighbours properties have a similar pontoon.

The Pontoon and gangway will be constructed and installed on site.

You have also applied for permission from the Marine Harbour Authority.

Works are planned to be conducted in one phase, over an estimated period of 2 days in the period April-May 2025.

Iteration 3

For this iteration we wanted to explore users needing a Defra account to send the exemption notification information. Part of the reason being enabling users to save and comeback later - for example if they needed to get coordinates or check on the dates of the activity.

The design switches from a linear flow to using the task list component. This enables users to see all sections that need completing upfront and complete them in any order they choose. Using this pattern would also benefit the more complex and lengthy standard marine licence and self-service marine licence journeys when those are designed and developed.

The prototype journey assumes the user already has a Defra account and Government Gateway account which the user signs in with.

Various questions in the 'Check if you need a marine licence' journey were updated with different or new content.

We no longer ask for name and email address as they are asked for during the account creation.

'Provide site details' (previously 'Location details') has been simplified. Users are first asked if they have site coordinates. If they don't, they're shown the map, if they do, they're asked whether they want to enter coordinates manually or upload them via a file. Previously, with the 3 options on that page, the novice users found the page daunting.

Hint text has been added to the project name page to help give clarity to why they are being asking for it.

The start and end dates pages now have some text explaining they should allow some extra time, otherwise they'll need to start the process again if they miss the given dates.

Project background and activity details text fields have been removed completely as legal said they weren't required.

A summary of the activity the user has picked from the 'Check if you need a marine licence' journey is played back on the Check answers page.

The file upload component was updated so it uses the latest improved version from the Design System.

The 'Defra account bar' with the account name is a placeholder borrowed from another Defra service. We'd possibly be looking to use the new Service navigation component from the Design System and incorporate the account links into that.

Iteration 4

This iteration is still being worked on, but the main changes are how we ask for site details - asking whether they want to draw on our map, rather than asking if they have the coordinates - and providing a support phone number.

The section will be updated when the changes are complete.
