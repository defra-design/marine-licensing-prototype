This prototype is made using the GOV.UK Prototype Kit and components and patterns from the the GOV.UK Design System.

User journeys worked on so far

  • Check if you need a marine licence
  • The exemption notification journey

Prototype limitations

If you see a page titled 'Journey end' it means either a feature you've selected isn't available or you need to go back and answer a question differently.

Drawing on a map

For drawing an area/site on a map, the prototype uses the map software that the 2024 alpha were using and developing. This has some limitations in terms of usability and accessibility and we have been investigating other fully accessible options that could be used instead, including the Flood risk map that is being developed within Defra.

Iteration 1

The exemption notification journey starting with Check if you need a licence questions that allows for answering questions relating to removal of sediment from the sea bed for scientific research.

Linear journey with no account creation / sign in required.

Pete, our Content Designer, worked on the Check if you need a licence questions making sure they followed the Government Digital Service style guide and restructuring the question format where necessary.

The Marine Protected Areas (MPA) question (/versions/2025-01-27/check/marine-protected-areas), links to an arcgis mapping tool which has various layers on it to help identify different areas. This is linked to from the live service, but in the prototype the map loads with only the MPA layer visible, rather than all of them, which is currently the case in live.

Out of the location details options the available ones are draw on a map and specify the centre of a square and its width.

Iteration 2

Based on the UR feedback some of the Check if you need a licence questions were updated with different or new content, but we also added in new pages to allow us to test a different scenario more appropriate for novice users - pontoon construction in a residential setting.

Features added:

  • File with coordinates upload options, with KML file upload the one that works
  • Specifying centre of a circle and it's width as the location
  • Manually entering multiple coordinates of the area
  • Specifying which coordinate system is used for coordinate entry
  • Check answers and review location now uses Cards style summary lists, with location changes happening only on the Review your location page with a single link to that page from Check answers

We didn't have time to iterate all the recommendations from all the UR feedback (specifically dates guidance and duplication of project and activity details), as we were waiting for a response from the legal team.

Iteration 3

For this iteration we wanted to explore users needing a Defra account to send the exemption notification information. Part of the reason being enabling users to save and comeback later - for example if they needed to get coordinates or check on the dates of the activity.

The design switches from being linear to using the task list component, which enables users to see all sections that need to be completed and to do that in the order they choose. Using this pattern would also be beneficial for the more complex standard marine licence and self-service marine licence when those come to be designed and developed.

The prototype journey assumes the user already has a Defra account and Government Gateway account which the user signs in with.

Various questions in the Check if you need a licence journey were updated with different or new content.

Provide site details (formerly location details) has been simplified to ask if the user has site coordinates and if not show them the map and if they do then ask whether they want to enter them manually or upload them in a file. Previously with the 3 options on that page the novice users found the page daunting.

Hint text has been added to the project name page and start and end dates pages.

Project background and activity details text fields have been removed completely as legal said they weren't required.

A summary of the activity the user has picked from the Check if you need a licence journey is played back on the Check answers page.

The file upload component was updated so it uses the latest improved version from the Design System.

Cancel links were added next to the Save and continue buttons but they became complicated as users could be coming from the task list or the review and check answers page. As time was limited these were taken out but we would need to include them for the developed service.

Being able to add multiple sites would also need to be an option. Early on designs were done in Figma, but those designs would need to be reviewed and updated.
