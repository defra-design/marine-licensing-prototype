Capital dredge

This is calculated by using the current depth of the sea or river bed and how much you need to deepen it by, for example if the current depth is 10m chart datum and you're deepening it to 12m chart datum, the dredge depth is 2m of sediment.

Maintenance dredge

This is the amount of sediment you need to remove to maintain the approved depth, for example if the approved depth is 12m chart datum and the sediment has built up so the current depth is 10m chart datum, the dredge depth is 2m.
