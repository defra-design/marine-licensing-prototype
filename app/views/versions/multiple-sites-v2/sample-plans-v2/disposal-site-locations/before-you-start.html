Before you start

Read the guidance on deposits. This includes guidance on disposal of material at sea.

In the next step, you'll need to tell us the following for every site:

  • if you want to dispose of material at an existing or new disposal site
  • type of material being disposed
  • method of disposal
  • maximum disposal volume for the full licence period
  • if any material being disposed of will be re-used to benefit the environment (beneficial use)

Using an existing designated disposal site

You need to tell us the location of the existing designated disposal site you plan to use. You can provide the existing site location by selecting it from a list. You can filter the list and search for an existing disposal site by using any, or all, of the following:

  • disposal site code
  • name of the disposal site
  • site location
  • marine area the disposal site is located in
  • site status

Request a new disposal site

To request a new disposal site to be designated you need to tell us its location.

The Marine Management Organisation will review the location you provide to determine if a new disposal site can be designated at this location.

You can provide the site location in 1 of 2 ways. You cannot do both. You can either:

  • upload a shapefile or KML file with the coordinates
  • enter the coordinates manually

Projects with multiple sites

You need to give separate location details for each disposal site if your project has multiple sites.

If you're uploading a file you need to include all the sites in 1 file.

Defining the site boundary

The coordinates must mark the full boundary of every site, including space for any movement or small changes in the activity location.
