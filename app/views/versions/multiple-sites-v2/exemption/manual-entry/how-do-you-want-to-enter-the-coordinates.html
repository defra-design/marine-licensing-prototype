Help with choosing how to enter coordinates

For a circular site

Choose this option if you only know the coordinates at the centre of the site. We'll use the coordinates and width you enter to create a circular area.

Make sure the circle covers the full area where the activity will take place, including any expected movement.

This example shows a 30-metre circular site using a single set of coordinates as the centre point.

For a site with multiple coordinates

Choose this option if your site has a clear boundary or defined shape. You'll need to enter coordinates in order, so they join to form a closed shape.

Make sure the shape includes the full area where the activity will take place, including any expected movement.

This example shows a site marked by 4 sets of coordinates to form a custom shape.
