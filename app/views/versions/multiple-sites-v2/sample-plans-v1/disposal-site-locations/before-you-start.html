Before you start

Read the guidance on deposits. This includes guidance on disposal of material at sea.

In the next step, you'll need to tell us the following for every site:

  • if you want to dispose of material at an existing or new disposal site
  • type of material being disposed
  • method of disposal
  • maximum disposal volume for the full licence period, you can also tell us maximum annual and campaign disposal volumes if you have this information
  • if any of the material is being disposed of for beneficial use

Providing new disposal site locations

You can provide the site location in 2 ways:

  • upload a file with the coordinates
  • enter the coordinates manually

Projects with multiple sites

You need to give separate location details for each disposal site if your project has multiple sites.

If you're uploading a file you need to include all the sites in 1 file.

Defining the site boundary

The coordinates must mark the full boundary of every site, including space for any movement or small changes in the activity location.
