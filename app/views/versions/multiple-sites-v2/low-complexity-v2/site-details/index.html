Before you start

In the next step, you'll need to tell us the following for every site:

  • site name
  • type of activity, for example construction
  • category of activity, for example maintenance construction
  • activity description
  • maximum duration of the activity
  • schedule of when the activity will take place
  • possible impacts of the activity and ways to reduce them

Providing site locations

You can provide the site location in 2 ways:

  • upload a shapefile or KML file with the coordinates
  • enter the coordinates manually

Projects with multiple sites

You need to give separate location details for each site if your project has multiple sites.

If you're uploading a file you need to include all the sites in 1 file.

Defining the site boundary

The coordinates must mark the full boundary of every site, including space for any movement or small changes in the activity location.
