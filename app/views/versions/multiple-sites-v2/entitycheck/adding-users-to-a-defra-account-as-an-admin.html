Adding users to a Defra account as an admin

Read guidance on how to invite users to your Defra account and give them access to Get permission for marine work.

How to add users to your Defra account as an admin

As an account admin you can invite and remove other team members or third parties and manage a Defra account. You can also give admin rights to other team members.

We recommend there are at least 2 account admins to prevent issues accessing and managing the account.

Adding users is a 2-step process. First you invite the user to join the account as a team member or intermediary user and give them account access.

Once they've registered you then assign them service access which will either be as an admin or standard user. Intermediaries can only be added as standard users and cannot access the account dashboard.

Adding a new user (account access)

  1. Go to your Defra account and select the Manage link next to the organisation account you want to add a user to.
  2. Select either Manage team access or Manage intermediary access.
  3. Select either Add a team member or Add an intermediary.
  4. Enter the email address of the new user and select Continue.

What happens next

  1. You'll be notified that an invitation has been sent to the user and that they must respond to it within 30 days before it expires.
  2. Select Continue.

Giving a new user service access

Once the new user is registered you need to give them access to the service.

You can either do this:

  • using the Give this person access to the use online services link in the email notification sent to you
  • through their profile page.

Using the email link to give a new user service access

  1. Select the Give this person access to the use online services link in the email notification sent to you
  2. This will take you to the user's profile page. Select Give account access.
  3. Choose if you want to give team members Admin or Standard user access. For intermediaries you can only select Standard user.
  4. Select Continue.
  5. Select the service Get permission for marine work even if this is the only available option then select Continue.
  6. Select the service role for the new user.

You'll then be notified that the user now has access. The new user will get an email confirming that their access is fully set up.

Giving a new user service access through the Defra account dashboard

  1. From the Defra account dashboard select Manage next to the organisation record.
  2. Select either Manage team access or Manage intermediary access.
  3. Select the user's name and select Give account access.
  4. For team members, choose either Admin or Standard user. For intermediaries you can only select Standard user.
  5. Select Give service access.
  6. Select the service Get permission for marine work even if this is the only available option then select Continue.
  7. Select the service role for the new user.

You'll then be notified that the user now has access. The new user will get an email confirming that their access is fully set up.
