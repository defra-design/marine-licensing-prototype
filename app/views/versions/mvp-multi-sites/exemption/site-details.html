Site details

Before you start

In the next step, you'll need to provide the following for every site:

  • a site name
  • the exact location of the site
  • the dates the activity will take place
  • a description of the activity

Providing site locations

You can provide site locations in 2 ways:

  • upload a file with the coordinates
  • enter the coordinates manually

If your project has multiple sites, you must give separate location details for every site. If you're uploading a file, you can include all the sites in one file.

Defining the site boundary

The coordinates must mark the full boundary of every site, including space for any movement or small changes in the activity location.

If the activity happens outside the marked boundary, it will not be covered by the exemption.
